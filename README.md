# 👩‍💻 주요 기능 
- 금융결제원 오픈뱅킹API, 카카오로그인API
- 구매자(문자인증API, 주소API, 아임포트결제API)
- 판매자(본인인증API, 그리드.js, 에디터, 공유하기API)
- 관리자(구글차트API,검색페이징기능)

### 　　
### 　　

# 💡 기능 요약
> ## 구매자
 - 후원하기 또는 리워드 선택 후 결제창 이동
 - 전화번호 정보 없을 시 문자인증 후 전화번호 등록
 - 우편번호 API를 이용해 주소 등록
 - 결제 API를 이용하여 결제
 - 결제성공 시 결제완료창 이동
 - 마이페이지 : 회원정보 수정 및 탈퇴, 내가쓴 댓글보기, 1:1 문의, 관심 프로젝트 목록, 내가 후원한 프로젝트 목록


### 　　
> ## [판매자](>##프로젝스-상세) 🙋(본인참여)
 - 프로젝트 상세 내용, 프로젝트 진행 현황
 - 좋아요 기능
 - url 복사, 페이스북, 트위터, 카카오 공유하기 기능
 - 판매자 공지사항 작성 🙋
 - 사용자 문의/응원/후기 댓글 작성 & 판매자 답글 작성 🙋
 - 프로젝트 및 댓글 신고하기 기능 🙋
 - 마이페이지 : 나의 프로젝트 목록, 주문배송 관리 기능
 - 금융결제원 오픈뱅킹 API를 이용한 계좌등록 기능 🙋

###
 - 프로젝트 만들기 : 사전 유의사항 체크 및 본인인증 🙋(본인참여), 프로젝트 정보, 일정, 스토리, 리워드, 정책, 판매자 정보 작성


### 　　
> ## 관리자
 - 관리자 페이지 메인 : 프로젝트 달성률 차트, 구매/판매/블랙리스트 회원비율 차트, 최근 신고 알림, 최근 승인요청 프로젝트 
 - 고객센터 : 공지사항, 1:1 문의, FAQ 게시판 목록 작성, 조회, 수정, 삭제
 - 프로젝트 승인 : 요청된 프로젝트 승인/반려 목록, 프로젝트 정보 상세보기 및 프로젝트 미리보기
 - 프로젝트 목록 : 승인된 프로젝트 진행중/마감/중지 목록
 - 정산이체 : 달성마감된 프로젝트 정산이체 기능 🙋(본인참여)
 - 신고목록 : 신고된 회원 및 프로젝트 목록, 블랙리스트 전환 기능
 - 회원목록 : 회원정보 상세보기, 구매/판매/블랙리스트 구분
